# 줄서 디자인 시스템

## 브랜드

- 서비스명: 줄서 (Julseo)
- 로고: 워드마크 "줄서" + 말풍선/체크 아이콘 조합
- 포인트 컬러: 블루 계열 (버튼, 활성 탭, 강조 텍스트에 사용)
- 배경: 화이트 기본, 콘텐츠는 카드형(둥근 모서리) 레이아웃

## 상태·의미 색상 규칙

전체 화면에 공통 적용하며, 프로세스 다이어그램과 동일한 규칙을 사용한다.

- 코랄/레드: 자동화가 필요한 구간 (예: 캠페인 자동등록, AI 추천 수신)
- 앰버/오렌지: 사장님이 직접 결정하는 구간 (예: 최종 선택)
- 블루: 외부 연동 데이터 (예: 예약 연동, 검증 프로필)
- 틸/그린: 완료 상태 (예: 체크인 확인, 신뢰 후기 확보)

이 규칙은 대시보드 배지, 프로세스 요약 패널, 폼 하단 태그칩 등 모든 화면에서 동일하게 사용한다.

## 내비게이션 구조 (IA)

- 최상단: 검색바(매장, 지역, 한산 시간대), 로그인, 회원가입, 광고문의(CTA)
- 메인 메뉴: 홈 / 캠페인 / 예약슬롯 / 방문인증 / 후기회수 / 오늘오픈 / 파일럿 / 광고문의
- 이 메뉴 구조는 핵심 기능 3가지(예약슬롯, 방문인증, 후기회수)를 그대로 1뎁스 메뉴로 노출한 것이므로, 향후 화면 설계 시 이 구조를 기준으로 삼는다.

## 홈 화면 구성 패턴

1. 히어로 배너 3종 (각각 이미지+배지+제목+설명)
   - "오프라인 방문 특화" 배지 - 한산 시간대를 방문 후기로 채우기
   - "자동 방문 인증" 배지 - QR·GPS 체크인으로 노쇼 감소
   - "후기 회수" 배지 - 체크인 후 후기 요청 자동화
2. 3단계 이용 순서 카드
   - 01 예약슬롯등록
   - 02 QR·GPS체크인
   - 03 후기자동회수
3. 카테고리 아이콘 목록(원형)
   - 오늘오픈
   - 한산시간
   - 맛집
   - 카페
   - 액티비티
   - 문화
   - 성수팝업
   - 광주파일럿
   - 부천파일럿
   - 운영가이드
4. 캠페인 리스트 + 필터 탭
   - 전체
   - 한산시간
   - 맛집
   - 카페
   - 액티비티
   - 문화
5. 우측 플로팅 CTA 카드
   - "사장님이신가요?"
   - 다크 배경
   - 서비스 요약 + 모집 등록 유도 링크

성수팝업, 광주파일럿, 부천파일럿은 PRD의 1차 타깃 지역 3곳과 대응한다.

## 폼 화면 구성 패턴

캠페인 등록 등 입력 화면 공통 패턴이다.

- 좌측: 섹션 번호 + 섹션 제목 + 부제(다음 단계 안내 문구) → 입력 폼
- 입력 폼: 2컬럼 그리드, 라벨은 입력창 위, placeholder로 예시 텍스트 제공
- 체크박스형 옵션은 제목+설명 조합으로 카드처럼 표시 (예: "시간대 분석 후 자동 등록 사용")
- 하단에 관련 태그칩 노출
  - 추천수신
  - 최종선택
  - QR·GPS체크인
  - 리마인드발송
  - 신뢰후기
- 우측: 전체 프로세스 요약 패널을 사이드에 고정 노출하여, 지금 입력 중인 단계가 전체 흐름 중 어디인지 실시간으로 보여준다.
- 프로세스 요약 패널 8단계
  - 사장님의뢰
  - 매장등록
  - 시간대분석
  - 자동등록
  - 추천수신
  - 최종선택
  - 체크인확인
  - 체험완료
- 프로세스 요약 패널 하단에는 상태 색상 범례를 포함한다.
