# 줄서 (Julseo)

줄서는 신규 오픈 오프라인 매장의 한산 시간대에 방문 인증된 체험단을 매칭해 초기 방문과 신뢰 후기를 확보하는 서비스입니다.

자세한 제품 요구사항은 [docs/PRD.md](docs/PRD.md)를 참고하세요.

## 팀 구성

| 이름 | 역할 |
|---|---|
| 김영란 | 팀장 |
| 정종환 | 팀원 |
| 위예은 | 팀원 |

## 기술 스택

| 영역 | 기술 |
|---|---|
| Frontend | Next.js |
| Language | TypeScript |
| Backend / Data | Firebase |
| Collaboration | GitHub |

## 핵심 기능

1. 한산 시간대 예약 슬롯 연동 캠페인 등록
2. QR·GPS 방문 인증 체크인
3. 후기 자동 회수·리마인드

## 주요 문서

- [PRD](docs/PRD.md)

## 현재 상태

리커리어에서 줄서 프로젝트로 피벗하며 저장소를 정리한 상태입니다. 구현 코드는 줄서 기준으로 새로 구성합니다.
